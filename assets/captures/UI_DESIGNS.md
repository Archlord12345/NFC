# Captures et Designs de l'Interface Utilisateur

Ce document décrit les principaux écrans de l'application NFC Cash, servant de référence pour l'interface utilisateur.

## 1. Écran de Connexion (Login)
- **Logo** : Icône NFC centrée en haut.
- **Titre** : "FINTECH" en gras, sous-titre "Secure. Fast. Effortless."
- **Champs** :
    - E-mail avec icône enveloppe.
    - Mot de passe masqué avec icône cadenas et bouton de visibilité.
- **Actions** :
    - Bouton "Login" avec flèche vers la droite.
    - Bouton "Use Biometrics" avec icône empreinte digitale.
    - Lien "Get Demo Account" pour les nouveaux utilisateurs.

## 2. Tableau de Bord (Wallet Dashboard)
- **En-tête** : Message de bienvenue personnalisé.
- **Carte de Solde** : Affiche le solde actuel (ex: 2,450.50 XAF) sur un fond dégradé.
- **Actions Rapides** :
    - Bouton "Recharger" (+)
    - Bouton "Scanner NFC" (icône scan)
- **Historique** : Liste des dernières transactions avec icônes (flèche rouge pour débit, verte pour crédit).

## 3. Scan NFC
- **Animation** : Cercle pulsant indiquant la recherche d'un tag NFC.
- **Instructions** : "Approchez votre appareil d'un tag NFC ou d'un autre téléphone."
- **Status** : Indicateur d'état (Scanning, Processing, Success/Error).

## 4. Historique des Transactions
- **Liste** : Vue détaillée de toutes les transactions passées.
- **Filtres** : Par date ou par type (Recharge, Transfert).
- **Détails** : Montant, Date, et ID de transaction.
